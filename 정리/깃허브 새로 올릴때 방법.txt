1. git add . -> git에 추가를 해주는작업
2. git status - > 파일을 커밋하는 작업
3. git commit -m "OOOOO" -> 케밋네임을 정해줘야함
4. git push origin master -> 깃에 보내는 작업

-------------------

git 이그노어는 내가 깃에 올리기 싫은 것을 안올릴수있다.
